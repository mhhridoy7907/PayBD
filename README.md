# PayBD 💳

[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)
[![Made with HTML](https://img.shields.io/badge/Made%20with-HTML-orange)]()
[![Made with CSS](https://img.shields.io/badge/Made%20with-CSS-blue)]()
[![Made with PHP](https://img.shields.io/badge/Made%20with-PHP-purple)]()

PayBD is a **web-based digital payment platform** designed to demonstrate online payment integration, wallet management, and transaction handling.  
This repository includes both **frontend and backend code** built with HTML, CSS, JavaScript, and PHP.

---

## 🚀 Features
- Send and receive money  
- Mobile recharge  
- Bill payment  
- Transaction history  
- Account balance check  
- Admin panel for user and transaction management  
- Responsive and interactive UI  
- Integration ready for payment gateways  

---

## 💻 Tech Stack
- **HTML** – Frontend structure  
- **CSS** – Design & styling  
- **JavaScript** – Interactive behaviors  
- **PHP / Laravel** – Backend logic  
- **MySQL** – Database
